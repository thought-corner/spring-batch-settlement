### 일별 정산 배치 시스템 구축

#### 비즈니스 시나리오

1. 매일 발생하는 대량의 주문 데이터가 있다고 가정하고 정산 주기가 도래한 건에 대해 수수료를 제외하고 가맹점주에게 정산해주는 자동화 배치 시스템을 구현한다.
2. 정산 정책은 주문 발생일로부터 7일 뒤에 정산해야 한다.
3. 매일 새벽 4시, 어제까지 구매 확정이 난 주문 건들을 모아 수수료를 제외하고 정산 테이블에 데이터를 생성한다.

#### 상세 요구사항

1. 대상 데이터 선정 - `ItemReader`
2. 정산 금액 계산 로직 - `ItemProcessor`
3. 데이터 저장 - `ItemWriter`
4. 자동 실행 - 매일 새벽 4시 실행
